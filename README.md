# Sistema de Login

Aplicação de autenticação desenvolvida com arquitetura cliente-servidor, utilizando Node.js, Express e PostgreSQL. O projeto implementa cadastro de usuários, login seguro com criptografia de senhas através do bcrypt, autenticação baseada em JWT e proteção de rotas para estudo de boas práticas de desenvolvimento backend.

Tecnologias
HTML5
CSS3
JavaScript (Vanilla)
Node.js
Express.js
PostgreSQL
JWT (JSON Web Token)
bcrypt
Funcionalidades
Cadastro de usuários
Login com autenticação
Criptografia de senhas com bcrypt
Geração de token JWT
Rotas protegidas por autenticação
Integração com PostgreSQL
Validação de credenciais
Objetivo

O objetivo deste projeto é praticar conceitos fundamentais de desenvolvimento Full Stack, incluindo:

Comunicação entre Frontend e Backend
Autenticação e autorização
Manipulação de banco de dados relacional
Segurança no armazenamento de senhas
Estruturação de APIs REST
